# reverseCinematic

Este diretório contém scripts para simulação dinâmica de robôs usando Python.

## Conteúdo

- `script.py` — Simulação de movimento robótico (cinemática)

## Como usar

1. Instale o pygame:
   ```bash
   pip install pygame
   ```
2. Execute o script:
   ```bash
   python script.py
   ```

## Objetivo

Permite visualizar e testar modelos cinemáticos de robôs em ambiente gráfico.

Ideal para experimentos práticos e validação de algoritmos de navegação.
