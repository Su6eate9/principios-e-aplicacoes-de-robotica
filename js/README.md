# js

Scripts JavaScript para a versão HTML dos exercícios de robótica.

- `main.js` — Inicialização e controle geral
- `astar.js` — Algoritmo A\*
- `grid.js` — Gerenciamento da grade
- `mapLoader.js` — Carregamento de mapas
- `ui.js` — Interface e renderização

Utilize junto com `index.html` e `css/style.css` para rodar os exercícios em navegador.
