# assets

Este diretório contém arquivos de apoio para simulações e exemplos da disciplina.

## Estrutura

- `examples/` — Mapas de exemplo para o Algoritmo A\*
  - `small.txt` — Mapa 10×10
  - `medium.txt` — Mapa 20×20
  - `large.txt` — Mapa 30×30

## Como usar

Carregue os arquivos `.txt` na aplicação web para testar diferentes cenários de navegação.

Os mapas usam 0 para célula livre e 1 para obstáculo.
